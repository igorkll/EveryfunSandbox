статические параметры блока:
	mesh - меш который будет использован для блока
	texture - текстура которая будет использована для блока
	shader - шейдер который будет использован для блока. если вы так же установите и текстуру то вы получите ее в шейдере в качестве параметра __texture
	allowMultimesh - разрешает или запрешяет использования мультимеша для этого блока. по умолчанию true

примечания:
	преобразования блока из статического в динамический и наоборот является изменением состояния

динамические параметры блока:
	__material - если блоку задан меш и он не находится в мультимеше то это будет его материал. может быть использовано для установки параметров шейдера
	__rigid_body - если ваш блок установлен как динамический то это будет self с типом RigidBody3D
	__data - данные вашего блока которые сохраняются после перезахода в игру и изменении состояния
	__state - данные вашего блока которые не сохраняются после перезахода в игру но сохраняются при изменении состояния

калбеки:
	__firstInit - вызывается один раз при первом создании вашего блока, затем при изменении состояния не будет вызван
	__init - вызывается при создании вашего блока и при изменении состояния
