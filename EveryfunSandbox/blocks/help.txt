статические параметры блока:
	mesh - меш который будет использован для блока
	texture - текстура которая будет использована для блока
	shader - шейдер который будет использован для блока. если вы так же установите и текстуру то вы получите ее в шейдере в качестве параметра __texture
	allowChunkmesh - разрешает или запрешяет чанкинг меша для этого блока. по умолчанию true

примечания:
	преобразования блока из статического в динамический и наоборот называется изменением состояния
	изменения состояния блока по сути пересоздает его ноду
	выгрузка чанка с ващим блоком равноценна его уничтожению
	загрузка чанка с ващим блоком равноценна его созданию

динамические параметры блока:
	__material - это материал если блоку задан меш и меш не пренадлежит чанку. может быть использовано для установки параметров шейдера
	__rigid_body - если ваш блок установлен как динамический то это будет self с типом RigidBody3D
	__data - данные вашего блока которые сохраняются после перезахода в игру, выгрузке чанка и изменении состояния
	__state - данные вашего блока которые не сохраняются после перезахода в игру и выгрузке чанка но сохраняются при изменении состояния

калбеки:
	__initData - вызывается один раз при первом создании вашего блока, не будет вызван при изменении состояния и при перезаходе в мир
	__initState - вызывается при создании вашего блока и при перезаходе в мир
	__init - вызывается при создании вашего блока и при изменении состояния
